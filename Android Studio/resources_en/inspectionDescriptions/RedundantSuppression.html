Докладывает об использовании
  • аннотации @SuppressWarning, или
  • строковый комментарий// noinspection, или
  • комментарий JavaDoc/** noinspection */
который может быть безопасно убран, т.к. проверка больше не применима в данном контексте.

К примеру: 

public class C {
 // символ уже в частном доступе,
 // на аннотация все еще присутствует
  @SuppressWarnings({"WeakerAccess"})
 private boolean CONST = true;
 void f() {
    CONST = false;
  }
}
